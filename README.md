# 📊 AI Student Impact — Analyse de données

Projet d’analyse exploratoire sur l’impact de l’intelligence artificielle générative sur les performances académiques des étudiants.

Ce projet a pour objectif de maîtriser les fondamentaux de l’analyse de données avec **Python** :
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📁 Dataset

- **Source** : AI Student Impact Dataset (Kaggle)
- **Taille** : 50 000 lignes × 16 colonnes
- **Variables clés** :
  - `Post_Semester_GPA`
  - `Skill_Retention_Score`
  - `Burnout_Risk_Level`

---

## 🧠 Objectifs du projet

- Comprendre la distribution des données
- Calculer les statistiques descriptives
- Analyser les quartiles, percentiles et IQR
- Détecter les tendances et anomalies
- Visualiser les relations entre variables
